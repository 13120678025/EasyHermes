/ 问教篇 /

常谈不厌，解惑答疑

Q: 双击启动无反应，或提示微软 Defender 危险警示？

A: 本软件由于将完备的 Python 微型内核完全打包，且具备本地文件修改和终端交互操作。在由于数字证书未经过微软高额商业认证时，部分 Windows Defender 会弹出未知警告，请点击“仍要运行”并放行。

Q: 未成功弹出网页 Dashboard 终端，如何处理？

A: 少数安全性控制苛刻的 Windows 系统会阻断双击跳转默认浏览器的行为。当您确认终端正在运行（未报错关闭）后，直接在现代浏览器中手动访问地址 http://localhost:7777 即可跨进控制大殿。

Q: 设备更换，我的聊天记录和记忆是否随之销毁？

A: 绝不。得得益于便携锁定文件的安全机制。您所有的聊天记录（`state.db`）、模型额度、记忆沉淀（`MEMORY.md`）全部随盘存放于同级的 `workspace/` 目录下。只需将此工程整体迁移，即在任何电脑上完美复活。

Q: 局域网物理设备无法联动，如何调试？

A: 确保您的计算机与 ESP32 设备连接在同一个 Wi-Fi 网络内。您可以通过在局域网内广播配置 `WEB_HOST=0.0.0.0` 环境变量，允许外部物理外设跨节点对计算机进行 HTTP 数据探针上行。
